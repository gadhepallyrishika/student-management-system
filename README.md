# Student Management System (Python)

This is a simple yet effective Student Management System developed using Python. The project demonstrates basic CRUD (Create, Read, Update, Delete) operations for managing student records.

## Features
- Add new student details
- View all student records
- Update existing student information
- Delete student records
- Simple menu-driven console interface

## Technologies Used
- Python
- Basic file handling / data structures (if used)

## Project Purpose
This project helps in understanding core programming concepts such as loops, functions, conditional statements, and CRUD operations. It is useful for beginners to learn how real-world data management systems work at a basic level.

## Learning Outcome
- Strengthened Python programming skills
- Understanding of CRUD operations
- Improved problem-solving and logical thinking
